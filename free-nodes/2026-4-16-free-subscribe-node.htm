一、什么是Shadowrocket？

Shadowrocket 是一款专为 iOS 平台打造的网络代理工具，功能强大且配置灵活，主要用于设置代理、实现加密流量转发，从而突破地域或网络限制，同时保障用户的上网安全与隐私。

其核心特点如下：

  • 协议支持广泛：支持 Shadowsocks、Vmess、Trojan、HTTP、HTTPS、SOCKS5 等多种协议；

  • 规则灵活：可自定义规则分流，按域名/IP/应用进行流量分类管理；

  • 流量统计详细：实时监控上传/下载速度，帮助用户精确了解网络使用情况；

  • IPv6支持完善：适配现代网络环境；

  • 兼容订阅功能：一键导入节点，动态更新配置文件，极大方便用户维护节点列表。

对于追求隐私、自由、性能的iOS用户来说，Shadowrocket几乎是必不可少的工具。

三、Shadowrocket的基础配置教程

1. 添加代理节点

Shadowrocket支持手动输入节点或通过订阅链接一键导入节点列表。推荐使用订阅链接方式，便于统一管理与自动更新。

方法一：手动添加

  • 打开Shadowrocket；

  • 点击右上角“+”图标 → 选择节点类型（如Shadowsocks、Vmess等）；

  • 填入服务器信息：地址、端口、密码、加密方式等；

  • 保存后即可看到节点出现在列表中。

方法二：使用订阅链接导入

  • 获取订阅地址（如Clash牧场或Telegram群中分享的）；

  • 打开Shadowrocket → 点击“配置” → 添加配置 → 输入订阅URL；

  • 命名后保存，并点击右上角刷新按钮同步节点。

多数节点提供商都会提供兼容Shadowrocket的专用链接。

2. 设置规则分流（可选）

Shadowrocket支持多种代理模式：

  • 全局代理：所有流量均通过代理；

  • 规则模式：按域名/IP或国家归属进行智能分流；

  • 绕过大陆地址：国内流量直连，境外流量代理。

你可以通过添加公共规则集（如ACL4SSR）来自动化管理：

  • 打开“配置”；

  • 点击“规则” → “远程规则”；

  • 添加远程规则地址（例如GitHub上的规则订阅）；

  • 启用该规则配置。

3. 启用代理

  • 配置完成后，返回主界面；

  • 点击右上角开关按钮，首次启用会提示添加VPN配置；

  • 点击“允许”，输入设备密码，授权后将自动建立VPN隧道；

  • 状态栏显示VPN图标，即表示代理已启动成功。

四、使用技巧与进阶配置

✅ 配合第三方DNS使用

推荐搭配 DoH（DNS over HTTPS）服务，如Cloudflare的 1.1.1.1，增强域名解析隐私性与准确性。

路径：设置 → DNS → 添加 → 选择 DoH → 输入地址。

✅ 绑定特定应用使用代理

可开启“分应用代理”模式，仅对部分App启用代理，如YouTube、Telegram等，节省流量并提高效率。

路径：设置 → 应用规则 → 添加 → 选择App → 指定代理方式。

✅ 实时测速与节点切换

长按某个节点可进行测速，数值越小表示连接延迟越低；推荐根据延迟情况选择速度最佳节点。

✅ 多配置切换

支持保存多个配置文件，适用于不同场景（如工作/娱乐/旅行），在主界面一键切换。

FAQ

Netch 如何实现 UDP 游戏加速？
Netch 支持 UDP 转发，可专门为游戏流量建立高速通道。结合策略组和节点延迟检测，可有效减少丢包和延迟，提高游戏体验的流畅性和稳定性。
Nekobox 导入 Clash 订阅后规则丢失，可能原因？
部分客户端在解析订阅时只保留节点信息而忽略 rules 或 proxies-group，建议使用支持完整解析的订阅转换器（如 sub-converter）先将订阅转换为兼容格式再导入，或手动合并规则文件。
Netch 如何为游戏流量提供加速？
Netch 支持 UDP 转发，可为游戏流量建立高速通道。结合策略组和节点延迟检测，可减少丢包和延迟，保证游戏访问稳定流畅，提高网络体验。
如何在 macOS 上运行 ClashX？
ClashX 是 macOS 平台专用客户端，用户可在 GitHub 下载 DMG 安装包。安装后导入订阅即可使用，支持系统代理集成、自动更新和延迟测试。界面简洁直观，深受苹果用户欢迎。
Sing-box 如何按 IP 段实现智能分流？
Sing-box 支持根据 IP 段匹配流量，将国内流量直连、国外流量走代理。结合策略组和规则文件，实现精细化流量管理，保证访问速度和稳定性，同时提高节点使用效率。
如何更新 Clash 的订阅节点？
在 Clash 中更新订阅节点，只需在配置页面中点击“Update”或“Refresh”按钮即可。系统会重新下载订阅链接中的内容，自动替换旧节点。这样能确保用户随时获取最新可用节点，避免连接失败或延迟过高。
Hysteria2 在丢包环境下如何配置更稳？
可调整 Hysteria2 的拥塞控制与重传参数，启用 FEC（前向纠错）并合理设置 MTU 与窗口大小以适应链路特性。还可尝试切换 QUIC 参数或更换中转以减少丢包影响。
Mihomo 如何启用 TUN 模式？
在 Mihomo 的配置文件中启用 `tun:` 模块并安装系统驱动即可实现全局代理。TUN 模式可将所有系统流量导向代理核心，实现真正意义上的全局加速。
Sing-box 如何兼容 Clash 的配置文件？
Sing-box 支持 Clash 规则格式的解析，用户可直接导入 Clash 的 YAML 配置文件进行使用。虽然部分高级参数需要调整，但大多数分流和策略配置都能自动识别，便于迁移。
Hiddify Next 用户流量配额如何设置？
管理员可为每个用户设置带宽和流量上限，超出限制时可降速或阻止访问。确保服务器资源合理分配，避免单用户占用过多带宽，提高整体网络公平性和使用效率。
